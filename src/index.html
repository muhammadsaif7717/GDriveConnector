Setup Google API Credentials

Enter your Google Cloud Project Client ID and Secret to allow this app to connect to Google Drive.

If you want to set up your Client ID and Secret, please read our Help page for step-by-step details.

Help & Documentation

How to Setup Google Drive Connector

To use this application, you need to create a Google Cloud Project and obtain a Client ID and Client Secret.

  1. Go to the Google Cloud Console.
  2. Create a new project.
  3. Navigate to APIs & Services > Library and enable the Google Drive API.
  4. Navigate to APIs & Services > OAuth consent screen:
    • Choose External and click Create.
    • Fill in the required App information fields.
    • Under Test users, add the email addresses of the users who will be testing this app.
  5. Navigate to APIs & Services > Credentials:
    • Click Create Credentials > OAuth client ID.
    • Select Desktop app (or Web application with redirect URI http://localhost:3000/oauth2callback) as the application type.
    • Copy your Client ID and Client Secret.
  6. Paste them into the Setup page in this app and click Save.

Note: As long as the OAuth consent screen is in "Testing" mode, only the email addresses you added as test users will be able to log in successfully.
