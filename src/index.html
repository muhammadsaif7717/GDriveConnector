Setup Guide

Follow these steps to generate a Client ID and Client Secret from Google Cloud.
This allows the app to securely connect to your Google Drive.

1

cloud Create a Google Cloud Project

  • arrow_right Go to the Google Cloud Console.
  • arrow_right Click Select a project > New Project...
  • arrow_right Enter a project name, then click Create.
  • arrow_right Click Select a project and choose the project you just created.
2

api Enable Google Drive API

  • arrow_right Open the side menu > Navigate to APIs & Services > Library.
  • arrow_right Search for Google Drive API.
  • arrow_right Click on it, then click Enable.
3

shield_person Setup OAuth Consent Screen

  • arrow_right Navigate to APIs & Services > OAuth consent screen.
  • arrow_right Click Get Started.
  • arrow_right Enter App Name G Drive Connector.
  • arrow_right Enter a support email, then click Next.
  • arrow_right Select External, then click Next.
  • arrow_right Enter a contact email, then click Next.
  • arrow_right Enable the I agree checkbox, click Continue, then Create.
4

key Generate Credentials

  • arrow_right Click Create OAuth client.
  • arrow_right Select Application type: Desktop app.
  • arrow_right Enter App Name G Drive Connector.
  • arrow_right Click Create, then click OK.
  • arrow_right Click on your new G Drive Connector client in the list.
  • arrow_right You will now see your Client ID and Client Secret. Copy and paste them into the Setup page of this app!
5

group_add Add Test Users

  • arrow_right Go to Audience from the sidebar in the Google Cloud Console.
  • arrow_right Under Test users, add the email addresses of the users who will be testing this app.
